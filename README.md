### Setting up:

clone project by pasting this command on terminal

`git clone git@github.com:eclairAB/irs-system.git`

go to the project directory and run;

`composer install`

after setting up your `.env` you can then install voyager;

`php artisan voyager:install --with-dummy`

after successful setup you can run `php artisan serve` to start the system!
